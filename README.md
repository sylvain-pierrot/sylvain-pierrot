# About Me

[![wakatime](https://wakatime.com/badge/user/068a54f9-253e-4c1c-b6bd-8c5eed569758.svg)](https://wakatime.com/@068a54f9-253e-4c1c-b6bd-8c5eed569758)

Hi, I'm Sylvain.

I started working in DevOps, mostly around infrastructure automation, reliability, and distributed systems. Over time I became more interested in what happens underneath the platforms themselves: networking, operating systems, and low-level software.

My work gradually shifted toward space and delay-tolerant networking software. I now spend most of my time building and experimenting with systems close to the OS: Linux networking, protocols, virtualization, and Rust.

I like understanding how things actually work rather than just using them.

---

## 🛰️ Current Interests

- Delay/Disruption Tolerant Networking (BPv7)
- Linux kernel & networking
- Rust systems programming
- Virtualization & microVMs
- Satellite communications & space networking
- Distributed systems in disconnected environments
